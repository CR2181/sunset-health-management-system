## 技术路线

MVP 阶段锁定：

```text
前端：HTML + CSS + 原生 JavaScript
后端：NestJS + TypeORM
数据库：MariaDB/MySQL，兼容 PostgreSQL
认证：JWT
```

不引入 Vue、React、Spring Boot、微服务、Kubernetes、消息队列等重型技术。

## MVP 边界

当前 MVP 做：

- 管理端基础展示。
- 登录认证。
- 后端模块骨架。
- 数据库持久化。
- 设备台账和设备事件预留。
- AI 事件预留。

当前 MVP 不做：

- 真实摄像头播放。
- 真实 AI 模型训练或推理。
- 真实硬件协议接入。
- 家属端独立应用。
- HIS、医保、监管平台深度对接。
